# FocusLab 角色画像与 IP 提示词

## 说明

- 来源：
  - `docs/01-产品概述与用户分析.md`
  - `prototype/screens/persona-card.html`
- 用途：整理 FocusLab 已有“正式用户画像 + 人格角色 + 组合角色”，便于后续为每类任务画像生成 IP 角色图。
- 提示词策略：
  - 默认面向角色立绘、角色卡、海报式半身像或全身像。
  - 关键词偏“科研拟人 + 潮流 IP + 高识别度道具 + 明确配色”。
  - 可直接用于 Midjourney、Flux、SDXL，建议再按你想要的画风追加风格词。

## 全局风格模板

### 通用正向提示词模板

`角色名, 科研拟人IP, stylized character design, expressive face, signature outfit, symbolic props, clean silhouette, premium illustration, sharp lighting, cinematic composition, detailed accessories, character sheet style, high detail`

### 通用负向提示词模板

`low detail, blurry, bad anatomy, extra fingers, duplicate body parts, messy background, unreadable props, text watermark, logo, cropped face, flat lighting`

## 使用建议

- 如果要做统一系列视觉，建议固定这些维度：
  - 画幅：角色卡 / 半身海报 / 全身设定图
  - 画风：二次元、扁平潮玩、轻赛博、学院幻想
  - 一致元素：FocusLab 标识、番茄钟、任务卡、时间刻度、论文纸张
- 如果要提高角色辨识度，建议在每次生图时固定：
  - 主色 1 个
  - 辅色 1 个
  - 核心道具 1 个
  - 动物/意象 1 个
  - 情绪标签 1 个
- 如果你要继续往下做，我下一步可以直接补一版：
  - “适配 Midjourney 的短提示词版本”
  - “适配即梦/可灵的中文短句版本”
  - “每个角色的头像版提示词”
