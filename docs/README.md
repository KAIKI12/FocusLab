## v1.1 变更摘要

基于全面审查,v1.1 补充了以下内容:

- 数据模型修复: 新增 daily_task_assignments 等4张表,完成率计算引入计划锁定机制
- 时区/跨天: 逻辑日概念,UTC存储规范,日期分界线,跨天边界处理
- 崩溃恢复: timer_state 持久化,启动恢复流程,日结算补算
- 新功能: FTUE(5步引导)、任务微复盘、重复任务、固定日程、后台任务、任务排序
- AI增强: 幻觉防护、多提供商适配、上下文管理、隐私确认
- 体验: 异常状态UI、极简模式、无障碍设计、休息结束三选一
- 工程: 内存预算、更新机制、日志系统、安全性设计、国际化预留
- 附录: 完整设置项清单(53项)、界面清单(59个)、关键流程图(6个)
- 路线图: Phase 1-3 各+1周,总周期约10-12周
